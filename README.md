* Responsive Design
* Dark & Light Mode
* Hero Section
* Character Showcase
* Game Modes Section
* Tournament Section
* Modern UI with Tailwind CSS
* Built with Next.js App Router

## 🛠️ Tech Stack

* Next.js
* React
* Tailwind CSS
* shadcn/ui

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open http://localhost:3000 with your browser to see the result.
